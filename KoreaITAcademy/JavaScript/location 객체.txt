location 객체 

: location 객체는 현재 브라우저에 표시된 HTML문서의 주소를 얻거나, 브라우저에 새 문서를 불러올 때 사용합니다.

href : URL 전체정보를 리턴, 특정페이지로 이동
protocol : 프로토콜 정보를 리턴
hostname : 호스트 이름과포트번호를리턴
pathname : URL 경로 부분의 정보를 리턴
